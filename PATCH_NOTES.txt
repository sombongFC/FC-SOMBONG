FC Sombong League v2.01

Fitur ditambahkan:
- Export Gambar Top Skor berupa poster PNG hitam/kuning.
- Copy Teks WA untuk share ke grup.
- Peringkat otomatis digabung jika jumlah goal sama.
- Catatan own goal tidak dicantumkan.
- Tetap membawa fitur foto pemain, import statistik awal, dan footer Team FC Sombong.

Cara deploy:
1. Upload seluruh isi folder ini ke root repository GitHub.
2. Render Root Directory dikosongkan.
3. Build Command: npm install
4. Start Command: npm start
5. Manual Deploy -> Clear build cache & deploy.
